# Dutch Fluency Coach Skill

Companion Skill for the Dutch Fluency MCP server.

This public Skill guides Claude to use Dutch Fluency as a learner-friendly coach for:

- Adaptive CEFR placement tests
- Dutch sentence correction with rule explanations
- Dutch Fluency certificate issue and verification
- Progress comparison between retakes
- KNM / inburgering practice

The Skill lives in [`dutch-fluency-coach/SKILL.md`](dutch-fluency-coach/SKILL.md).

MCP server: `https://mcp.dutchfluency.com/mcp`

This Skill does not replace the MCP server and does not include private curriculum data. It only teaches Claude a simple public workflow and safety guardrails for using the tools.
